Top 10 Interview FAQs (Quick Revision Sheet)

Interview Questions

1. What is Web API?
Answer
A RESTful service used to expose data and business functionality over HTTP.
2. What is REST?
Answer
REST is an architectural style that uses HTTP methods to perform CRUD operations on resources.
3. What is the difference between API and Web API?
Answer
API is any interface for communication, while Web API specifically communicates over HTTP.
4. What are HTTP Methods?
Answer
GET = Read, POST = Create, PUT = Update, PATCH = Partial Update, DELETE = Remove.
5. What is the difference between PUT and PATCH?
Answer
PUT replaces the entire resource, while PATCH updates only specific fields.
6. What is Dependency Injection (DI)?
Answer
Dependency Injection automatically provides required services to classes, reducing tight coupling.
7. What is Middleware?
Answer
Middleware is software that processes HTTP requests and responses in the ASP.NET Core request pipeline.
8. What is Attribute Routing?
Answer
Attribute Routing defines API endpoints using attributes like [Route] and [HttpGet].
9. What is Model Validation?
Answer
Model Validation ensures incoming request data is valid using Data Annotations and ModelState.
10. What is JWT Authentication?
Answer
JWT uses a signed token to securely authenticate and authorize users without server-side sessions.
